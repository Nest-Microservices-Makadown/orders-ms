EXCELENTE!

## Dev Steps 

1. Clonar repo
2. Instalar depencias:
```bash
$ npm install
```
3. Crear archivo `.env` basado en `env.template`
4. Correr docker compose 
```bash
$ docker compose up -d
```
5. Ejecutar la app 
```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
